Dev log 4 — Amazon Alexa and friends

Hey everybody. This week has been filled with so much tech and writing. That aside I would like to discuss two main side projects in this devlog.

Amazon Alexa skills and Wikicite

Both of these side projects lead into a genre of software that I really have not done before. Alexa skills with bot building and Wikicite with web scraping.

Alexa skills

Amazon Alexa is the personal assistant behind the Amazon Echo and Echo Dot. Skills are basically the apps that run on Alexa.

Today, I was able to build my first Alexa skill. I call it Russian Facts. It is just a simple skill that tells the user facts about Russia.

As I get more familiar with the SDK and framework, I hope to build stronger and powerful apps. I also hope that these skills will transfer over to chat bot building.

Wikicite

Wikicite was a web scraping project I started working on last year. Sadly, I got sidetracked and was forced to stop midway.

Wikicite is going to be a tool that takes Wikipedia articles and shows you just the citations and their reliability. This will depend on both manual and automated criteria.

In the continuation of this project, I hope to hone my web scraping tools and learn some data science along the way.

That’s the main side projects I’m working on. If you liked this dev log, please be sure to like, follow, and share.

Michael Navazhylau is a Software Engineer in St.Louis. He works on fun software projects every week and helps out at IT2GO2. You can check out his website at Mechasparrow.
