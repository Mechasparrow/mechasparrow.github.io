Switching over to Angular 2 and starting on my new project

So today I started using Angular 2. It actually reminds me a bit of react. Which is a little odd. This is mostly due to the fact that Angular has migrated over to using the component software pattern. This is also what react uses as well. For the new project that I started working on yesterday, I decided to use Angular 2.

The (AI) bot project

The project is about building a bot with natural language processing and support for extension. I am building the UI with electron. I hope to learn more about machine learning and speech recognition while working on this project. I am also using JavaScript because of its broad and diverse ecosystem. If this does not go well I thinking I might just start trying to build bots or chat bots anyway. I believe that bots will provide a new level of automation to users. Though this will depend on the tasks that the bot is asked to perform.
