Dev log #17 — FreeCodeCamp Data Visualization post-mortem

Hey, everybody. It’s Michael again. This week, I have spent time working hard to get my Data Visualization Certificate from FreeCodeCamp.

I was able to complete my certification yesterday. So in this dev log, I would like to focus on the work that went into it.

Certification requirements

There is 10 coding challenges required for the Data Visualization Certificate.

The first five of these challenges are React based.

The last five of these challenges are D3.js based.

React Challenges

React is a front-end framework for building UIs in JavaScript. It is a framework designed by Facebook and used by many web developers.

The React challenges for this certificate included

  • Build a Markdown Previewer
  • Build a Camper Leaderboard
  • Build a Recipe Box
  • Build the Game of Life
  • Build a Roguelike Dungeon Crawler Game

The first three of these challenges were easy. It was the last two that were especially difficult.

This was because the feedback and the debugging period was longer. This was due to the graphic manipulation and setup that had to be done.

Because the last two projects relied on some sort of pixel-like grid, I used styled HTML tables to solve the challenges.

D3 Challenges

D3 is a JavaScript library that allows manipulation of SVG and HTML elements based on data. This makes it good for the creation of graphs and other data visualizations.

The D3.js challenges included

  • Visualize Data with a Bar Chart
  • Visualize Data with a Scatterplot Graph
  • Visualize Data with a Heat Map
  • Show National Contiguity with a Force Directed Graph
  • Map Data Across the Globe

None of these D3 challenges were difficult in particular. The main challenging part was actually learning D3 itself.

The concepts that D3 introduced were confusing and hard to understand at first. As I continued completing the challenges they became easier to grasp though.

Conclusion

FreeCodeCamp tries very hard to put its students through a rigorous learning program. They make it very difficult to cut corners, and they make sure that you know your stuff.

If you wish to be a full stack web developer, I would definitely recommend FreeCodeCamp.

It has helped fill several holes in my web development knowledge.
