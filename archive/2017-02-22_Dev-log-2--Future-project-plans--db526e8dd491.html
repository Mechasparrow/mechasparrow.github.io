Dev log 2 (Future project plans)

Another week of software development has passed. I’ve completed a basic version of my local social network app called HackerProf. I have also submitted my side projects to 1ppm. I’ll cover some information about HackerProf and my plan for this week.

HackerProf

HackerProf is a local social network tool/Social Engineering tool. Currently it allows you to add contacts and keep logs of them. In the next version, I hope to add social network pulling. This means you can pull data from their public social media profiles such as Facebook and Twitter.

Swipe File

This week I plan to start working on my swipe file project. A swipe file is a collection of excerpts of content and other writing. This is a project to help writers (and me) create swipe files of content that they find on the web. This could also be useful for journalists as well.

Cleanup duty

I will also make sure to complete any uncompleted projects on Github. I will also tie up any loose ends regarding my internet presence somewhat. (Broken links, old information etcetera)
