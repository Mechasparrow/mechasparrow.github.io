How to build a Desktop Application with Electron and React

Hey, everyone. It’s Mechasparrow again. I’ve been working with Electron alongside other front-end technologies such as React. I had some trouble setting the projects up.

I decided that I should write a tutorial to help others who might be trying the same thing.

You can view the full source code for the tutorial at https://github.com/Mechasparrow/electron-react-tutorial

Alright, let’s start!

Setting up React

We are going to install and setup a simple react project first. Here are the commands.

npm install -g create-react-app
create-react-app your-app
cd your-app

Adding Electron Code

We only need one file to make this Electron compatible. This will be called main.js. We will add it to the root of the project directory.

// ./main.js
const {app, BrowserWindow} = require('electron')

let win = null;

function createWindow() {
  // Initialize the window to our specified dimensions
  win = new BrowserWindow({width: 1000, height: 600});

  // Specify entry point
  win.loadURL('http://localhost:3000');

  // Show dev tools
  // Remove this line before distributing
  win.webContents.openDevTools()

  // Remove window once app is closed
  win.on('closed', function () {
    win = null;
  });
}


app.on('ready', function () {

  createWindow();

});

app.on('activate', () => {
  if (win === null) {
    createWindow()
  }
})

app.on('window-all-closed', function () {
  if (process.platform != 'darwin') {
    app.quit();
  }
});

We have it load the URL at http://localhost:3000 because that is where the react server will be running.

Modifying the package.json

In order for electron to properly find the app the package.json needs to be modified. Make sure to add this to the package.json.

{
  ...
  "main": "main.js",
  ...
}

Install Electron

We can install electron globally by running this command through npm

npm install -g electron

After that we are ready to run the project!

Launching the Project

In order for us to run the project, we will need to do two things. The first is launching the react server. The second is launching electron.

First,we run

npm start

to start the react server.

In a separate window, we launch electron by running.

electron .

That should do it! I hope you enjoy your desktop application development with Electron and React :)
