Dev log #13 — FreeCodeCamp and Emojilog

Hey, everyone. It is dev log #13 (summer edition). This week I worked with FreeCodeCamp, Emojilog, TheSwipe.

I’ll dive into more details about each project.

FreeCodeCamp

FreeCodeCamp is a free online coding boot camp for full stack web development.

I did receive a full stack certification from Lynda.com, but there was not much code involved.

FreeCodeCamp helps you learn how to develop software interactively. Currently, I’ve obtained my Front End Certification from FreeCodeCamp.

I still have Data Visualization, Back End, and Full Stack to complete.

Emojilog

I started adding experimental features to Emojilog. The feature I added this week was Sentiment analysis.

As mentioned before, this is where the software can extract emotion from text. This was simple to add.

This is because there was already a JavaScript library for it called sentiment.

You can check out the latest version at http://emojilog.surge.sh/

TheSwipe

I finally finished creating TheSwipe. TheSwipe is a ruby on rails application for research. I created it to organize web resources and text snippets. You can check it out at https://theswipe.herokuapp.com/

Whats next?
