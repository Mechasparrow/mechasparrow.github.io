Wikicite — Wikipedia viewer focused on links with reliability

Have you ever done a paper that requires research, but they told you that you can’t use Wikipedia? Well, Wikicite is designed to circumvent that issue.

What’s the next best alternative to using Wikipedia? Using the research that went into those articles of course! Wikicite helps you find the links from those articles and presents them with how reliable each link is.

Wikicite is the React Web application that views links from Wikipedia articles. You can check it out at wikicite.surge.sh for the latest build. You can also check out the source code at https://github.com/Mechasparrow/wiki-cite.

So far I’ve managed to achieve the link extracting and viewing for it with the API. All I need to do now is actually pipe it through a prioritization algorithm.

I’m currently using an approach that is similar to what I used with morality machine in which I used a sort of white-list approach. I’m simply using a JSON file for the whitelist.

Hopefully, I should get a more polished release out sometime in April. For now, check the current progress on GitHub and the build.
