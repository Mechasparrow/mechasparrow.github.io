Dev log #19 — Getting work done and new ventures

Hey, everybody. It’s Michael Navazhylau a.k.a @mechasparrow. It has been such a fun week filled with both business and creative endeavors. This was FreeCodeCamp, St. Louis Website Development, and Hammercircuit related work.

FreeCodeCamp (Back-End)

I’ve been leveling up and building my developer skills with FreeCodeCamp. It is a free online resource for learning full-stack web development.

If you have any interest in learning how to program, I would recommend it.

This week, I was able to complete the last API project. This means that I am halfway completed with the back-end certification.

St. Louis Website Development

stlwebdev is a website development agency that I started last week with a friend of mine.

We finished the first version of the website yesterday. We also did some conference calls and planning this week.

You can check out the website at http://www.stlwebsitedevelopment.com/.

We will start working with clients this month.
