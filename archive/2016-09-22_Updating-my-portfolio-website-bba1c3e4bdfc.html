Updating my portfolio website

I just completed the major revision to my website. You can check out at mechasparrow.github.io . The main thing that I decided to do with this website is that It would provide all the necessary information upfront. I decided to do this with my website.

Old website versus new website

The old website was built at a time just when I was getting the hang of web design. This made the website very experimental and look weird. The colors I chose did work well together. After building some more websites, I was able to look at the website with a better perspective. This allowed me to build a portfolio website that’s straight to the point, yet conveys what needs to said. The only issue that still may need to be resolved is how the projects are organized.

The new website was build with simplicity in mind. I made sure all the important content could be viewed at the front. What really make the new website different is the fact there is no blog on there anymore. The blog has been transferred to medium. This allows for non-blog important info to be on the website.

Well that should be it as far as the website goes. See you around
