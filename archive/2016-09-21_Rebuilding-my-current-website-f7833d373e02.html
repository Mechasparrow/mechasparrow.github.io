Rebuilding my current website

Portfolio websites don’t need to be overly complicated however they should convey the essential information. The last portfolio website had several holes in it that I decided I would fix. The great part is I can completely rip out the blogging part and just link to medium.

I have decided to design a cover website like this template version that bootstrap has built which you can check out right here. I hope to rebuild it a fashion that I can proud of.

In order to build the website I am currently using Jekyll. It is a static website generator. I enjoy using this tool due to it support for partials, markdown, and other template tools.
