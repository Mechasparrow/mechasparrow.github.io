Dev log #15 — StartupGen

Hey, everyone. It’s another dev log. Today, I’m going to talk about StartupGen. It’s a side project that I created that got onto the front page of Product Hunt a couple of days back.

I’ll go into more detail about the creation of that and then I will talk about development for the next week.

How did StartupGen start?

The idea of StartupGen occurred when I was reading a post by Eric Stromberg. It was called the Startup Idea Matrix.

After reading the article, I took a look at the Spreadsheet and realized that you could make a generator out of it.

How did I make it

I built it by converting the Startup Idea Matrix into a CSV and then into 2 separate JSON files. I then used ReactJS to render a template that picked a random item from both of the JSON files.

Deployment

I created a simple logo with Figma. I then went over to Product Hunt and submitted the link along with the logo and screenshots. I then went to sleep.

The following day, I opened up Product Hunt and saw that I was on the front page. I realized again I forgot to install analytics, so I set it up on there.

What’s next?

Well, I’ll add a couple new features to StartupGen. This would include new styling and name generation. I will also continue with my FreeCodeCamp curriculum.

Thanks for reading the dev log. If you liked it, please be sure to follow, recommend, and share.
