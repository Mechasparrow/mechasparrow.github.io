Launch of cybernotes 1.0.0

Yeah almost the end of year. Survived Finals and all the things in between. That’s for different post. I would like to discuss the application called cybernotes that I developed during the first week of winter break. You can check it out here.

What is cybernotes?

Cybernotes is a web note platform for writing, storing and sharing notes. It currently has support for creating text and markdown notes. If you have a note that you want to share. You can make the note public and send the link to a friend or colleague.

Why use cybernotes?

Cybernotes can be used to help unclutter your note space. Say you have notes in a class that are taking up space. You could take a picture of them and access it anywhere. Listening to a college lecture? Use cybernotes to take notes from the professor. And store links to resources for the class.

Future of cybernotes

Some future features of cybernotes include

  • storing pictures
  • note collections
  • Searching notes
