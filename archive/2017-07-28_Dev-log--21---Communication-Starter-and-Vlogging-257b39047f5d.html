Dev log #21 — Communication Starter and Vlogging

Hey, everyone. It’s Mechasparrow. This week, I have not really been able to put in much development time because of vacation.

I was able to build a prototype for the side-project I was working on. This side project is called Comm-start. I’m also vlogging

The Communication Starter

This project kind of stems from personal experience. Sometimes, I find that it is hard to start conversations with others.

I’m also very interested in the Wikipedia API. This application is basically just a generator application with added features. It generates a topic that you can talk about with links to stay relevant and informed.

So far it only links to the Wikipedia articles. It also is really basic right now.

You can view comm-start at http://commstart.surge.sh/
