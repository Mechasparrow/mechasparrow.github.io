How to build a Desktop Application with Electron and Angular

Hey, everyone. It’s Mechasparrow again. I gained quite a bit of interest from the previous web development tutorial called How to build a Desktop Application with Electron and React. I have decided to make an Angular version of that tutorial.

You can view the full source code for the tutorial at https://github.com/Mechasparrow/electron-angular-tutorial

Alright, let’s start!

Setting up Angular

We are going to install and setup a simple angular project first. Here are the commands.

npm install -g @angular/cli
ng new your-app  
cd your-app

Adding Electron Code

We only need one file to make this Electron compatible. This will be called main.js. We will add it to the root of the project directory.

// ./main.js
const {app, BrowserWindow} = require('electron')
let win = null;
function createWindow() {
  // Initialize the window to our specified dimensions
  win = new BrowserWindow({width: 1000, height: 600});
  // Specify entry point
  win.loadURL('http://localhost:4200');
  // Show dev tools
  // Remove this line before distributing
  win.webContents.openDevTools()
  // Remove window once app is closed
  win.on('closed', function () {
    win = null;
  });
}

app.on('ready', function () {
  createWindow();
});
app.on('activate', () => {
  if (win === null) {
    createWindow()
  }
})
app.on('window-all-closed', function () {
  if (process.platform != 'darwin') {
    app.quit();
  }
});

We have it load the URL at http://localhost:4200 because that is where the angular server will be running.

Modifying the package.json

In order for electron to properly find the app the package.json needs to be modified. Make sure to add this to the package.json.

{
  ...
  "main": "main.js",
  ...
}

Install Electron

We can install electron globally by running this command through npm

npm install -g electron

After that we are ready to run the project!

Launching the Project

In order for us to run the project, we will need to do two things. The first is launching the angular server. The second is launching electron.

First,we run

npm start

to start the angular server.

In a separate window, we launch electron by running.

electron .

That should do it! I hope you enjoy your desktop application development with Electron and Angular :)
