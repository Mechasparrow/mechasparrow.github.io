How I made a fun generator app in React

A couple days, I built startup-gen, a web application for generating startup ideas. It is based off the Startup Idea Matrix by Eric Stromberg. I read through his article, and shortly afterward, I decided to build a generator based off of it. It has resulted in some funny and interesting combinations. I’ll discuss some key important steps that went into this project.

File tree

Here’s an overview of the project structure for the react-application. I used create-react-app as a template.

Getting that tasty JSON

First I downloaded the spreadsheet as a CSV. I then used a simple spreadsheet editing program to separate it into 2 separate files for the 1st row and column. After that, I ran both of the files through csvtojson. I finally ran the command

csvtojson startup-services.csv > json/startup-services.json
csvtojson startup-markets.csv > json/startup-markets.json

This resulted in the JSON files being generated. This allowed for the data to be easily navigated.

Main React Code for the Generator

This code basically selects a random market and random feature and puts it into a template by using a variant of the random function. “The startup offers x for y” is an example of the template. React is really good for these types of applications since it has its own templating language called JSX. The message changes in the state variable of the component each time the button is pressed. If you would like to see the full source code, check out startup-gen at https://github.com/Mechasparrow/startup-idea-gen.
