Dev Log #24 — VR, Live Streaming, and Haskell!

Dev Logs are back!

Yo. We’re back with official dev logs again! I’ve been working on 3 side-projects mostly. These side-projects are Virtual Reality, Haskell, and Live Streaming.

Let’s dive into each of these projects.

Virtual Reality

I have been working on this WebVR application called VRWorld for about a week now. VRWorld is WebVR exploration game.

I’m writing the WebVR application with a library called A-Frame. It allows you to build VR applications that run inside your browser with HTML and JavaScript.

In VRWorld, the player will explore a series of rooms while interacting with the environment.

Vague, I know.

Haskell

A friend of mine recently introduced me to Haskell, a functional programming language. I have begun learning it with the intention of building a web application with it.

The back end will be written in Haskell. The front end will be written in Elm, a front end programming language.

Live Streaming

I recently received an email from LiveEdu. They are a live streaming service for programming related content. This has reminded me of the live streams I used to do on their service and Twitch.

Starting this week, I will be live streaming once again.
