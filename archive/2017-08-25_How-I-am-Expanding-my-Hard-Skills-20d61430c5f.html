How I am expanding my hard skills

and how you can too!

Hello, everyone. I’m in a really cool class.

What’s cool about it? Well, it is an experiential learning class. It means that you learn by doing.

In this class called Spark, we learn and train for careers that we are interested in. One of the requirements for the course is that we must put in hours towards hard skill development.

Let’s talk about some ways to develop these hard skills.

What are hard skills?

Hard skills are teachable skills that can be defined and measured. This means that they are fairly tangible and easier to validate.

Online courses

One way to obtain hard skills is by enrolling in online courses. Some websites that offer online courses are Coursera, Udacity, and edX.

These courses are often backed by large companies or universities. This makes them very credible.

Fieldwork

Doing work in the field is another great way to build hard skills. Some examples of field work are internships and project work.

Each internship or project requires picking up new skills to complete the tasks.

Side Projects

Doing a project for fun is also another great way to pick up hard skills. This is because often times you will have to learn along the way to complete the project.

You want to build a game or write an app? Well, you need to learn the tools and skills necessary to complete them.

Many of the hard skills that I have picked up over the years have been a direct result of side projects.
