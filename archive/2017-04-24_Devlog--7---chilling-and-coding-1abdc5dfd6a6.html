Devlog #7 — chilling and coding

Hey, everybody. Sorry for the delay. Lots of tests in my high school right now. I’m pulling through so it’s not so bad. That is the reason that this dev log got delayed for couple of days. I’ll go over some things I’ve been working on.

Emojielog

I’ve talked about this before in a previous post. I created this tool called emojielog that a joke project. You describe each day by one or a set of emojis. I have a sort of prototype version developed. You can see it at https://emojielog.glitch.me/. Its nothing much right now. I’ll try to attach user authentication to it and redo it off of glitch.

Glitch stuff

As seen from the previous topic, I’ve been using glitch quite a bit recently. This is due to its versatility and back-end features. It allows for building any full-stack JavaScript app for free online. If you’re interested in Javascript or Web development, I would suggest checking it out.

Spark Tech

I’ve recently gotten accepted into Spark! This is a program that my school district offers to high school students. Luckily, I will be attending the program for my Junior year (year 11) of high school. This will help me achieve things that I would not be able to meet in a high school setting. For more information on how cool Spark! is check out http://parkwayspark.com/

I hope you enjoyed this development log. Please make sure to heart, share, and follow. Sorry if it seemed rushed. I’m trying to push through the end of my high school year right now.
