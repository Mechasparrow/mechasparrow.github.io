Dev log #14 — Dungeon Crawler with React

Hey, everybody. This is dev log #14.

This week I worked with FreeCodeCamp and React. The main project I worked on this week was a dungeon crawler game.

I am currently attending ACT prep. Because of this, I have decided to focus on increasing my programmer knowledge for the time being.

Dungeon Crawler

One of the assignments of FreeCodeCamp was to build a dungeon crawler game with reactjs. This took me some time because of the amount of functionality I had to implement.

To save time and not worry too much about graphics, I used the table element provided in HTML. The whole game world also ended being a large 2-dimensional array as well.

You can see the full game and source code at https://codepen.io/marioluigiman/pen/EXajZM

Whats next?

For next week, I will be continuing with the FreeCodeCamp projects. I may also create a small project that parsed Awesome lists from GitHub.
